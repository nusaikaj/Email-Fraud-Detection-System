# Email Fraud Detection


This is a simple **Email Fraud Detection web application** built using **Python, Machine Learning, and Flask**.  
The system analyzes email text and predicts whether the email is **Fraud/Spam** or **Legitimate**.

---

## Technologies Used
- Python  
- Flask  
- Machine Learning  
- NLP (TF-IDF)  
- Naive Bayes Classifier  

---

## How It Works
1. Email text is taken as input from the user  
2. Text is converted into numerical features using TF-IDF  
3. A trained Naive Bayes model predicts the result  
4. The result is displayed on a web page
